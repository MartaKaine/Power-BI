# 📊 Power BI Data Analysis Project (Bootcamp)

This project was completed during my **Data Technician Bootcamp** and focuses on analysing and visualising data using **Microsoft Power BI**.

The project demonstrates how raw data can be transformed into **interactive reports and dashboards** that help organisations understand trends and make data-driven decisions.

---

## 📁 Project Overview

In this project, I used **Power BI Desktop** to transform, analyse, and visualise datasets.

The workflow included:

1. Importing data from external sources
2. Cleaning and transforming data using **Power Query**
3. Creating calculated fields using **DAX**
4. Building interactive reports and dashboards

The final reports allow users to explore data dynamically using filters and slicers.

---

## 🛠 Skills Demonstrated

### Data Transformation (Power Query)

Data preparation was performed using **Power Query**, including:

- Cleaning and shaping datasets
- Removing unnecessary columns
- Changing data types
- Combining and transforming data sources

Data preparation is a crucial step in building reliable data reports.

---

### DAX Calculations

Used **DAX (Data Analysis Expressions)** to create:

- Calculated columns
- Measures for summarising key metrics
- Aggregated values used within visualisations

These calculations help extract meaningful insights from the data.

---

## 📊 Interactive Reports

Power BI reports were designed to allow users to interact with the data using:

- **Slicers**
- **Filters**
- Cross-filtering between visuals
- Drill-down capabilities

This allows users to explore the data from multiple perspectives.

---

## 📈 Visualisations Created

A variety of visualisations were used to present insights clearly:

- **Bar charts** – category comparisons
- **Line charts** – trends over time
- **Pie charts** – category distributions
- **Maps** – geographic analysis
- **Tables and KPI visuals** – summarising key metrics

These visualisations help communicate insights in a clear and engaging way.

---

## 📖 Data Storytelling

A key goal of this project was **data storytelling**.

The reports were designed to:

- Highlight key trends
- Provide intuitive navigation for users
- Present insights in a clear and structured format

This allows decision-makers to quickly understand patterns within the data.

Power BI is one of the most widely used **business intelligence tools** in industry.

This project demonstrates my ability to:
